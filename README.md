# מערכת שיבוץ משמרות (Streamlit)

אפליקציה לשיבוץ עובדים למשמרות על בסיס:
- רשימת עובדים
- דרישות משמרות (כמה עובדים צריך בכל יום/משמרת)
- העדפות עובדים (עדיפות לכל יום/משמרת)

בנוסף: יש מסך התחברות + רישום (SQLite) כך שרק משתמשים רשומים יכולים להיכנס.

---

## התקנה והרצה

```bash
pip install -r requirements.txt
streamlit run app.py
